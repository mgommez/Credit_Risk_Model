# Credit_Risk_Model
This is the final project -or thesis- of my dual bachelor university degree, particularly of the Business Administration side.

1. Relevant analyses and transformations of the dataset to facilitate the construction of the model
2. Implementation of a predictive model of loans' defaults - predicts whether a loan will default or not
3. Validation of the model's results

Dataset extracted from Kaggle,  https://www.kaggle.com/code/devraai/credit-analysis-default-prediction/input.
